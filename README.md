# gfree-libc
这是一个获取任意切换libc版本和源码级别调试能力的仓库
gfree-libc可以用来做什么？
==========================
gree-libc可以使你获取任意libc版本源码级（.c级别）的调试能力，你只需要gclibc指令。

我希望你的gdb使用的是pwndbg插件，因为其他插件可能会出现错误。
